# 🏔️ RUTA 9 | THE SUPER APP (v3.0)

Este es el núcleo de la **Super App de Ruta 9**, diseñada con una arquitectura de alto estándar (Next.js 15 + GSAP + Tailwind) y enfocada 100% en la experiencia móvil.

## 🚀 Características Implementadas:
1.  **Explorador Estilo "Stories":** Navegación horizontal fluida (`MenuExplorer.tsx`) con snap-scrolling para una experiencia tipo App nativa.
2.  **Dashboard "The Cockpit":** Sistema de indicadores arriba (Fuel Gauge para fidelización) y navegación táctil abajo (`Navigation.tsx`).
3.  **Cards Dinámicas:** Cada burger (B1 a B24) tiene su propia vista full-screen con ingredientes y precio dinámico.
4.  **ADN Visual:** Colores extraídos del logo (`#D1232B` Rojo, `#5D3A2C` Marrón) aplicados en un modo oscuro profundo ("Midnight Road").
5.  **Gamificación Ready:** Sistema de "Fuel" y XP integrado visualmente para compartir en redes.

## 🛠️ Cómo Lanzar:
1.  `npm install`
2.  `npm run dev`
